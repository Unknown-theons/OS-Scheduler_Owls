Shortest Remaining Time First (SRTF)

Preemptive CPU Scheduling Algorithm

About SRTF

Shortest Remaining Time First (SRTF) is a preemptive version of Shortest Job First (SJF) algorithm. In this scheme:

  • The process with the smallest remaining burst time is selected for execution
  • The running process can be preempted by a new process with shorter burst time
  • Provides optimal average waiting time for a given set of processes
  • May lead to starvation of processes with longer burst times
